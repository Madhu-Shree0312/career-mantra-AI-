## üìù AI Resume Analyzer

### Uploading Your Resume

#### Method 1: File Upload

1. **Click "AI Resume Analyzer" Card**

2. **Click "Choose file" Button**
   - Or drag and drop your file

3. **Select Your Resume File**
   - Supported formats: PDF, DOC, DOCX, TXT
   - Maximum size: 5MB

4. **File is Uploaded**
   - You'll see the filename
   - Click "Remove" to change file

#### Method 2: Paste Text

1. **Scroll to "Paste Resume Text" Section**

2. **Copy Your Resume**
   - From your document
   - Select all text (Ctrl+A / Cmd+A)
   - Copy (Ctrl+C / Cmd+C)

3. **Paste in the Text Area**
   - Click in the text box
   - Paste (Ctrl+V / Cmd+V)

### Analyzing Your Resume

1. **Click "Analyze Resume" Button**
   - Button turns purple with loading animation
   - "Analyzing..." text appears

2. **Wait for Results**
   - Usually takes 5-10 seconds
   - AI is analyzing your resume

3. **View Your Results**
   - **Score:** Overall rating (0-100)
   - **Detailed Analysis:** Strengths and weaknesses
   - **Improvement Suggestions:** Specific tips

### Understanding Your Score

| Score | Rating | Meaning |
|-------|--------|---------|
| 80-100 | Excellent | Resume is strong, minor improvements |
| 60-79 | Good | Solid resume, some areas to improve |
| 40-59 | Fair | Needs significant improvements |
| 0-39 | Needs Work | Major revisions required |

### Tips for Better Results

‚úÖ **Include:**
- Contact information
- Work experience with dates
- Education details
- Skills section
- Achievements with numbers

‚ùå **Avoid:**
- Personal photos
- Irrelevant information
- Spelling/grammar errors
- Unprofessional email addresses
- Excessive length (keep to 1-2 pages)

3. **Click "Generate Career Roadmap"**

   - Takes 5-10 seconds

4. **View Your Roadmap**
   - **Journey Header:** From ‚Üí To visualization

## üîß Troubleshooting

### Common Issues

**Problem: Can't Login**
- ‚úÖ Check email and password spelling
- ‚úÖ Try "Sign Up" if you haven't registered
- ‚úÖ Clear browser cache and cookies
- ‚úÖ Try a different browser

**Problem: AI Not Responding**
- ‚úÖ Check internet connection
- ‚úÖ Refresh the page
- ‚úÖ Try again in a few seconds
- ‚úÖ Check if backend is running

**Problem: Resume Upload Failed**
- ‚úÖ Check file format (PDF, DOC, DOCX, TXT)
- ‚úÖ Ensure file is under 5MB
- ‚úÖ Try pasting text instead
- ‚úÖ Check file isn't corrupted

**Problem: Roadmap Not Generating**
- ‚úÖ Fill in required fields (marked with *)
- ‚úÖ Check internet connection
- ‚úÖ Try with simpler role names
- ‚úÖ Refresh and try again

**Problem: Sidebar Not Showing (Mobile)**
- ‚úÖ Look for hamburger menu (‚ò∞)
- ‚úÖ Tap in top-left corner
- ‚úÖ Rotate device to landscape
- ‚úÖ Try on larger screen

### Error Messages

**"Failed to connect"**
- Backend server is not running
- Check internet connection
- Contact support

**"Invalid email or password"**
- Check credentials
- Try "Sign Up" if new user
- Reset password (if available)

**"Analysis failed"**
- Try again
- Check resume format
- Paste text instead of uploading
